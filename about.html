🔬 Algorithm Details

⊕

XOR Cipher

O(n) · Symmetric

Each pixel's RGB channels are XORed with the key value. XOR is a bitwise operation: if both bits are same → 0, else → 1. Because XOR is its own inverse (A⊕K⊕K = A), the same operation both encrypts and decrypts.

Encrypted[i] = Pixel[i] XOR Key
Decrypted[i] = Encrypted[i] XOR Key
✅Fast & simple
✅Perfect reversibility
⚠️Weak against known-plaintext attack
⇄

Pixel Swap (Fisher-Yates Shuffle)

O(n) · Position-based

All pixels are treated as a flat list and shuffled using the Fisher-Yates algorithm with a key-seeded random number generator (Python's random.Random(key)). The pixel values remain unchanged but their positions are scrambled.

indices = Fisher-Yates shuffle(seed=key)
encrypted[indices[i]] = original[i]
✅Preserves pixel values
✅Visually unrecognizable
⚠️Histogram unchanged
∑

Mathematical Operations

O(n) · Modular Arithmetic

Arithmetic operations (add, subtract, multiply) or bitwise operations (AND, OR) are applied to every pixel channel with modular wraparound at 256. The inverse operation recovers the original values.

Add: E[i] = (P[i] + K) mod 256
Subtract: E[i] = (P[i] - K) mod 256
Multiply: E[i] = (P[i] × K) mod 256
✅Multiple operation types
✅Value-based transformation
⚠️Multiply requires odd key
RGB

Channel Shift

O(n) · Color-space

RGB channels are rotated (e.g., R→G, G→B, B→R) and their values are independently shifted by key multiples. This combines positional channel scrambling with value modification, producing visually dramatic results.

R_enc = (G + key) mod 256
G_enc = (B + key×2) mod 256
B_enc = (R + key×3) mod 256
✅Dramatic visual effect
✅Dual transformation
⚠️Color histogram visible
∞

Arnold's Cat Map

O(n²) · Chaotic Dynamical System

Introduced by Vladimir Arnold in the 1960s, the Cat Map is a chaotic area-preserving map on a torus. Each pixel at (x, y) is mapped to a new position using the matrix transformation below, applied over multiple iterations. It's periodic — after enough iterations the image returns to its original state, making decryption possible by applying the inverse transform.

[ x' ] [ 1 1 ] [ x ] (mod N)
[ y' ] = [ 1 2 ] [ y ]

Inverse: x = 2x' - y' (mod N)
y = -x' + y' (mod N)
✅Highly chaotic scrambling
✅Based on ergodic theory
✅Used in academic research
⚠️Periodic (finite period)

🛡️ Cybersecurity Concepts

🔐

Symmetric Encryption

The same key is used for both encryption and decryption. Our algorithms follow symmetric key cryptography, like AES in concept.

🌀

Confusion & Diffusion

Shannon's principles: Confusion (obscure key-ciphertext relationship) via XOR; Diffusion (spread plaintext statistics) via pixel swapping.

📊

Histogram Analysis

A strong encryption should flatten the image histogram. Pixel swap preserves histogram; XOR and math ops change it significantly.

🔢

Key Space

With 256 possible keys, brute force is feasible. In production systems, keys would use 128-bit or 256-bit spaces (AES standard).

🎲

Pseudorandomness

Pixel swap uses Python's Mersenne Twister PRNG seeded with your key — deterministic but statistically random output.

∞

Chaos Theory

Arnold's Cat Map demonstrates how deterministic chaotic systems create apparent randomness, a foundation of modern cryptography research.
