Encryption Algorithms

Choose from 5 powerful pixel manipulation techniques

⊕

XOR Cipher

Each pixel value is XORed with a secret key. Fast, simple, and perfectly reversible — the same operation encrypts and decrypts.

Symmetric Lossless O(n)
⇄

Pixel Swap

Pixels are shuffled into random positions using a seeded pseudorandom permutation based on your key. Looks completely scrambled.

Position-based Key-seeded Fisher-Yates
∑

Math Operations

Apply arithmetic operations (Add, Subtract, Multiply, Bitwise AND/OR) to every pixel value with modular arithmetic for wrapping.

Modular Multi-op Custom

Channel Shift

RGB color channels are rotated and their values are shifted independently, creating a visually distorted color-space transformation.

RGB Rotation Value Shift Color-based
∞

Arnold Cat Map

A chaotic map from ergodic theory. Iteratively scrambles pixel positions using a nonlinear matrix transformation — highly secure.

Chaotic Iterative Non-linear
